# API Design for Food Delivery System

## Cart Management

- Add to cart
  - api/v1/cart/addItem
  - [itemInfo, customerId]
  - 201 Created
- View Cart
  - api/v1/cart/view
  - customerId
  - 200 OK & Cart Content
- Clear Cart
  - api/v1/cart/clear
  - customerId
  - 204
- Remove item from cart
  - api/v1/cart/removeItem
  - itemId, customerId
  - 204
- Update Quantity
  - api/v1/cart/updateQuantity
  - itemInfo, customerId
  - 204
